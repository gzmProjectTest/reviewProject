## 可配置属性与控制的功能

#### 属性说明：
|属性名 | 类型 | 必填 | 默认值 |说明 |
|---|---|---|---|---|
|options| array | 否 | [] | 选择项 |
|placeholder | string | 否 | 请输入 | 默认值 |
|value | string | 否 | - | 选择的索引，默认不需要传 |
|error| bol| 否 | false | 展示错误的，label变红 |
|label | string | 否 | - | 表单名|
|change| func | 否 | - | 值改变后的回调  {value: 0} value中的数组值代表的是索引。|
|mode | string| 是 | selector | 选择器模式 |
|title | string | 否 | 设置 | 选择器标题（仅安卓有效）默认值为 "设置" |
|range-key	|String|		|当 options 是一个二维 Array.<object> 时，通过 range-key 来指定 Object 中 key 的值作为选择器显示内容|


时间选择器：mode = time

|属性名	|类型	|默认值	|说明|
|------|----|-------|---------|
|value	|String|		|表示选中的时间，格式为 "hh:mm"|
|start	|String|		|表示有效时间范围的开始，字符串格式为 "hh:mm"|
|end|	String|		|表示有效时间范围的结束，字符串格式为 "hh:mm"|


日期选择器：mode = date
|属性名	|类型	|默认值	|说明|
|------|----|-------|---------|
|value|	String|	当前日期|	表示选中的日期，格式为 "YYYY-MM-DD"|
|start|	String|		|表示有效日期范围的开始，字符串格式为 "YYYY-MM-DD"|
|end|	String|		|表示有效日期范围的结束，字符串格式为 "YYYY-MM-DD"|
|fields|	String|	day|	有效值 year、 month、 day，表示选择器的粒度|

fields有效值：
|值	|说明|
|---|---|
|year|	选择器粒度为年|
|month|	选择器粒度为月份|
|day|	选择器粒度为天|

省市区选择器：mode = region

|属性名	|类型 |	默认值 |	说明 |
|------|----|-------|---------|
|value|	Array	|[]	|表示选中的省市区，默认选中每一列的第一个值|
|custom-item	|String	|	|可为每一列的顶部添加一个自定义的项|
|bindchange	|EventHandle|		|value 改变时触发 change 事件，event.detail = {value}， 暂不支持统计用区划代码（code）、邮政编码（postcode）。|
|title|	String	|	|选择器标题（仅安卓有效）默认值为 "设置"|

多列选择器：mode = multiSelector
|属性名	|类型|	默认值|	说明|
|------|----|-------|---------|
|range|	二维 Array/Array.<object>	|[]	|mode 为 selector 或 multiSelector 时，range 有效。二维数组，长度表示多少列，数组的每项表示每列的数据，如[["a","b"], ["c","d"]]|
|range-key	|String|		|当 options 是一个二维 Array.<object> 时，通过 range-key 来指定 Object 中 key 的值作为选择器显示内容|
|value|	Array|	[ ]	|value 每一项的值表示选择了 options 对应项中的第几个（下标从 0 开始）|
|bindcolumnchange| function | - |某一列的值改变时触发 columnchange 事件，event.detail = {column: column, value: value}，column 的值表示改变了第几列（下标从0开始），value 的值表示变更值的下标 |
|cusvalue|Boolean|false|mode 为 multiSelector 时，cusvalue 为 true 是可以通过slot为 single-picker组件自定义显示值|

### 异步多级请求

```javascript { .theme-peacock }

Page({
/* eslint-disable */
    data: {
        values:[],
        areas: [
            [
                {
                    name: '请选择'
                },
                {
                    name: '北京'
                },
                {
                    name: '天津'
                },
                {
                    name: '河北'
                }
            ],
            [
                {
                    name: '请选择'
                }
            ],
            [
                {
                    name: '请选择'
                }
            ]
        ]
    },
    selectorChange2: function(e) {
        console.log(e);
    },
    columnchange: function(e) {
        this.getOptions(e.column, e.value);
    },
    getOptions: function(column, value) {
        console.log(column, value);
        const areas = this.data.areas;
        const values = this.data.values;
        if (value === 0 || column + 1 === areas.length) {
            return;
        }
        for (let i = 0; i < areas.length; i++) {
            if (i > column) {
                areas[i] = [{
                    name: '请选择'
                }];
            }
        }
        values[column] = value;
        this.setData({
            areas,
        });
        this.mockData(areas[column][value].name, column).then(res=>{
            console.log(res);
            res.unshift({
                name: '请选择'
            });
            areas[column + 1] = res;
            this.setData({
                areas
            });
        });
    },
    mockData: function(name, key) {
        console.log(name, key)
        let a = key === 0 ? '地区' : '街道';
        return new Promise(resolve=>{
            setTimeout(()=>{
                let value = [];
                let i = 0;
                while(i < 10) {
                    value[i] = {
                        name: `${name}${a}${i}`
                    };
                    i++
                }
                resolve(value)
            }, 200)
        })
        
    }
});
```
